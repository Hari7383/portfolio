Beginner-level-playground

The Hari7383/Deep-Learning repository serves as a beginner level playground for exploring the core principles of deep learning, with a strong focus on image classification tasks. It includes foundational scripts and notebooks that demonstrate how different neural network architectures (like ANNs and CNNs) can be used to recognize and classify image data — a critical application of deep learning in real-world use cases.

Artificial Neural Network (ANN) Implementation (ann.py)

  • This script showcases a simple multi-layer perceptron (MLP) model using a dataset like MNIST or custom image inputs.
  • It walks through essential steps: input normalization, dense layer creation, activation functions (like ReLU), and backpropagation for training.
  • Great for understanding the basics of how neurons are connected and learn patterns.

Convolutional Neural Network (CNN) Implementation (cnn.ipynb)

  • CNNs are powerful because they can extract spatial hierarchies of features from images using convolutional layers, pooling, and fully connected layers.
  • This Jupyter notebook is focused on building CNN architectures — ideal for image recognition tasks.
  • Likely includes examples of model training, loss/accuracy plotting, and testing on datasets like CIFAR-10 or MNIST.

Dropout Technique (dropout.py)

  • Introduces dropout regularization, a method to reduce overfitting by randomly "dropping" neurons during training.
  • This improves generalization and model robustness, especially with small datasets.

Handwritten Digit Classification (neural_network_for_handwritten_digits_classification.py)

  • Shows the end-to-end pipeline: loading data, preprocessing, model creation, training, and evaluating performance
  • A complete mini-project to classify digits using a neural network, likely using the MNIST dataset.

Dataset Folder

  • Contains datasets or links used in the above scripts — possibly CSVs or image folders — to run the models with real input.
